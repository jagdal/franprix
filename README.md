## Description

Franprix Admin Dashboard est une application web interne destinée à l’administration et à la gestion des activités au sein du bureau.

Elle permet de centraliser les informations, suivre les projets, gérer les commandes et consulter les alertes en temps réel.

---

## Fonctionnalités

- Gestion des projets
- Suivi des commandes (palettes, produits, etc.)
- Tableau de bord administratif
- Consultation des alertes et notifications
- Visualisation des données en temps réel
- Interface simple et efficace pour usage interne
